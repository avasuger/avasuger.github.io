[Github博客搭建教程]

Diana 2022/8/10

😎安装软件

  1. 创建Github账号
  2. 创建库
    ![[Pasted image 20220810170122.png]]

安装并配置Node.js

安装完后打开CMD输入
node -v以及npm -v查看是否安装完成

  • tips

    可以在Node.js文件夹创建缓存文件夹
    >npm config set prefix”路径”
    >npm config set cache”路径”

  1. 下载Git及Hexo

    测试git –version

hexo下载:

npm install hexo-cli -g
hexo v

⚠️如果报错

报错解决

  1. 给予Node.js文件夹修改权限
  2. 找到Node.js文件夹里面的npm和npm.cmd文件
  3. 文本编辑两个文件找到prefix -p 改成prefix –location=globa
  4. 删除C:\Users\（你的用户名）\下的.npmrc文件

📐生成SSH

  • 随意进入一个文件夹
  • 右键找到Git Bash Here
  • ssh-keygen -t rsa -C “你的邮箱地址”(敲四次回车)
  • 找到密钥位置C:\Users\（你的用户名）.ssh\id_rsa.pub 公钥文件
  • 文本编辑全选复制
  • 进入Github
    ![[Pasted image 20220810192559.png]]
  • 找到SSH and GPG keys
    ![[Pasted image 20220810192740.png]]
  • 标题随意,公钥放Key栏(不要删除哪怕一个空格)
    ![[Pasted image 20220810192827.png]]
  • 测试是否绑定成功

    ssh -T git@github.com
    ![[Pasted image 20220810193237.png]]

🍴生成本地博客

创建文件夹,在文件夹右键git bash here
成功后文件夹会多出一些文件

hexo init # 初始化hexo

生成

hexo g # 生成
hexo s # 启动服务

http那串链接就是本地博客了
![[Pasted image 20220810194021.png]]
此时可以Ctrl+C停止

🍭生成网络博客

  • 进入Github仓库

  • 点击Code复制HTTP链接
    ![[Pasted image 20220810194410.png]]

  • 找到博客文件夹的_config.yml文件

  • 在最后面加入

    ![[Pasted image 20220810194649.png]]

  • 安装hexo-deployer

    npm install hexo-deployer-git –save

  • 生成页面

    hexo g

  • 上传Github(此操作极有可能因为网络而错误)

    hexo d

  • 最后会出现绑定页面,绑定即可
